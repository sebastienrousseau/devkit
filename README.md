# Devkit

Developer tooling and environment configuration for the Sebastien Rousseau ecosystem.

## Overview

Devkit provides scripts, configurations, and tools to streamline development workflows across the ecosystem.

## Contents

- `/scripts` - Automation scripts for common tasks
- `/templates` - Project scaffolding templates
- `/configs` - Editor and tool configurations
- `/hooks` - Git hooks for quality enforcement

## Installation

```bash
curl -fsSL https://raw.githubusercontent.com/sebastienrousseau/devkit/main/install.sh | bash
```
